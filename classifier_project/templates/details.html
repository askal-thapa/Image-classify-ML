The Analysis Pipeline

A visual journey of how we transform a raw image into a forensic verdict.

Step 1

Image Acquisition

The user uploads a high-resolution image. At this stage, the image contains millions of pixels and potentially large amounts of metadata.

Input: Any Size Format: RGB
Step 2

Preprocessing

The system intelligently resizes the image to 224x224 pixels. This standardization is crucial for the neural network. We also normalize pixel values from 0-255 to the expected range for EfficientNet to speed up mathematical convergence.

Resize: 224x224 Normalize
Step 3

Feature Extraction

Our EfficientNetB0 engine applies hundreds of convolutional filters. These filters act like "digital eyes," scanning for:

  • Unnatural edge artifacts (common in GANs)
  • Pixel interpolation inconsistencies
  • Lighting gradients that defy physics
Conv2D Swish Activation
94%
Step 4

Forensic Verdict

The extracted features are flattened into a single vector and passed through a Dense layer. A Sigmoid function squashes the result into a probability between 0 (AI) and 1 (Real).

Sigmoid Binary Classification
